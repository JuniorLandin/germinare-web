NPM version: 20.11.0

Para instalar e iniciar a aplicação, basta executar os comandos abaixo, de dentro da pasta inicial:

1) npm install --force
2) npm run dev

* Problemas com a dependência "react-bootstrap"?
 Execute o comando abaixo:
npm install --legacy-peer-deps
